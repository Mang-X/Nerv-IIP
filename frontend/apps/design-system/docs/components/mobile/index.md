# PDA 移动组件

`@nerv-iip/ui-mobile` —— 车间手持终端的组件库，触控 + 手势为主。左侧按**分类**浏览，每个组件都有独立页面与实时演示（手机宽度预览）。

- **基础**：Button、Badge、Cell、ListRow
- **表单**：Input、Checkbox、Radio、Switch、Stepper、SearchBar、Picker、DatePicker、ScanBar
- **导航**：TabBar、Tabs、Steps、NavBar、Collapse
- **反馈**：ActionSheet、BottomSheet、Dialog、Toast、NoticeBar、Result、Empty
- **手势 / 操作**：Fab、Grid、PullRefresh、SwipeCell

> 想一次看全部？查看 [完整示例总览](/components/mobile/gallery)（设备外观框中的沉浸式演示）。
