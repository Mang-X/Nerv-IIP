# PDA 移动组件

`@nerv-iip/ui-mobile` —— 车间手持终端的组件库，触控 + 手势为主。左侧按**分类**浏览，每个组件都有独立页面与实时演示（手机宽度预览）。

- **基础**：NvMobileButton、NvMobileBadge、NvCell、NvListRow
- **表单**：NvMobileInput、NvMobileCheckbox、NvMobileRadioGroup、NvMobileSwitch、NvStepper、NvSearchBar、NvPicker、NvMobileDatePicker、NvScanBar
- **导航**：NvTabBar、NvMobileTabs、NvMobileSteps、NvNavBar、NvMobileCollapse
- **反馈**：NvActionSheet、NvBottomSheet、NvMobileDialog、NvMobileToast、NvNoticeBar、NvMobileResult、NvMobileEmpty
- **手势 / 操作**：NvFab、NvMobileGrid、NvPullRefresh、NvSwipeCell

> 想一次看全部？查看 [完整示例总览](/components/mobile/gallery)（设备外观框中的沉浸式演示）。
