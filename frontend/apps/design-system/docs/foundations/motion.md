# 动效

动效服务于**反馈**与**连续性**，不喧宾夺主。统一的缓动令牌让全站节奏一致。

## 缓动令牌

| 令牌 | 曲线 | 用途 |
|---|---|---|
| `--ease-out-quart` | `cubic-bezier(0.25, 1, 0.5, 1)` | 通用进入 / 悬停 |
| `--ease-out-expo` | `cubic-bezier(0.16, 1, 0.3, 1)` | 滑动吸附、抽屉、回弹 |
| `--ease-in-out-quart` | 对称 | 标签页 / 指示器滑动 |

> 约定：`--ease-out-back` 令牌不存在，需要回弹时用 `var(--ease-out-back, cubic-bezier(0.34,1.4,0.64,1))` 的回退写法。

## 拖拽手感：橡皮筋

所有涉及拖动的组件共享一条**橡皮筋阻尼曲线**（`abs(x)^0.92 * 0.7`），保证手感一致：

- 范围内 **1:1 跟手**；
- 拖过边界时**阻尼回弹**（越拉阻力越大）；
- 松手按距离 / 速度决定吸附或弹回，过渡用 `ease-out-expo`。

应用于：移动端**侧滑操作**、**抽屉上拉**、**下拉刷新**。

## 微交互原则

- 触摸反馈优先用**背景 / 不位移**的属性，少用 `transform: scale`（会造成布局位移）。
- 退出动画要和进入同样讲究——避免"看起来没有动画"的突兀消失。
- 覆盖层用半透明 + `backdrop-blur` 的玻璃质感，克制不过分。
- 尊重 `prefers-reduced-motion`：所有过渡都有 `reduce` 降级。
