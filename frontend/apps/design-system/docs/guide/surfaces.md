# 三个表面

同一套令牌与设计语言，三种原生形态。

## 🖥️ 桌面 PC

控制台主表面：信息密度高、键鼠为主。语义令牌 + 12 色动态板，Pro 组件（DataTablePro、描述列表、时间线、图表）。克制动效，专业感优先。

→ [桌面组件](/components/desktop/)

## 📟 移动 PDA

车间手持终端：触控为主、单手操作、强光环境。原生质感控件，完整手势体系——侧滑操作、下拉刷新、抽屉拖拽关闭，统一的**橡皮筋阻尼**手感。安全区适配，≥44px 触控目标，覆盖层带玻璃质感。

→ [移动控件](/components/mobile/)

## 📊 一体机看板

工位大屏：远距离可读、触控操作、减少路径。工单大卡、报工步进、节拍图，为"少点几下完成报工"而设计。

→ [看板组件](/components/board)

---

## 一致性原则

- **令牌共享**：颜色、间距、圆角、动效缓动全部来自同一套 `theme.css`。
- **手感原生**：每个表面遵循其平台的交互基线，不强行统一手势或尺寸。
- **组件分层**：桌面/一体机用 `@nerv-iip/ui`，移动用 `@nerv-iip/ui-mobile`，但视觉血统一致。
