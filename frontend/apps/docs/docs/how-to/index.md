# 操作指南

操作指南按"角色 + 任务"组织，每篇回答一个具体问题：某个角色要完成某件事，按哪串页面、
什么顺序、注意什么。它和教程的区别：教程带新手第一次走通一条主线，操作指南给已经上手的人
按任务查步骤。

## 现在怎么用

本象限的正文随功能批次同步补齐，当前先这样查：

1. 到[按角色入门](/roles/)找到你的角色，路径表列出每条任务路径、页面串和当前可用性。
2. 三篇教程里的"操作步骤""常见失败/空态"章节，已经覆盖主线任务的操作要点：
   - [基础资料到工程资料与生产版本](/getting-started/engineering-to-production)
   - [需求计划到完工入库](/getting-started/planning-to-finished-goods)
   - [仓储收发与库存闭环](/getting-started/wms-inventory-cycle)

## 首批规划

首批操作指南按六个角色的第一周路径拆篇（见各角色页路径表），路径状态为"可用"的先写，
"部分可用/缺口"的等对应 GitHub issue 闭环后再补。文档缺口按季度统一盘点回收。
