# 参考

参考象限是页面与字段字典：按业务域列出页面入口、字段含义、单据状态与编号规则，供工作中
查阅，不承担教学。

## 现在怎么查

参考正文随功能批次同步补齐，当前先这样查：

- 页面入口与所属业务域：[核心业务流程图](/processes/)每张图下方的映射表，列出了页面路由
  与对应后端能力的对照关系。
- 每条主线的单据流与状态变化：三篇教程的"业务对象/单据流""状态变化"章节。

## 首批规划

- 页面目录：按业务域（主数据/工程/计划/MES/仓储库存/质量/设备维护/条码/ERP/审批）列出
  全部页面入口与用途。
- 字段字典：列表与表单中的关键字段含义、必填规则与取值来源。
- 单据状态机：各单据允许的状态与迁移动作。
- 编号规则：`WO-`、`WC-` 等业务编号的构成与分配方式。
