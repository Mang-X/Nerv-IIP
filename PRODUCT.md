# Product

## Register

product

## Users

工厂一线的**操作工、班组长、工艺工程师、设备运维**，以及平台管理员。使用场景是车间与办公室长时间盯屏作业：
- **桌面 PC**：控制台主表面，键鼠为主、信息密度高，工单 / 设备 / 质检 / 排产的管理与监控。
- **车间一体机（看板）**：工位大屏，远距离可读、触控操作、减少操作路径。
- **移动 PDA**：手持终端，单手 + 手势、强光环境、≥44px 触控目标。

他们要完成的"活"是具体的生产动作——下发/报工/质检放行/设备点检/排产，而不是浏览营销内容。

## Product Purpose

Nerv-IIP 是**数字工厂的控制平面（MES control plane）**。平台先行、业务后置：先把工单、设备、质检、排产等控制能力做稳，行业语义作为领域扩展叠加。设计的成功标准是让人**更快、更准、更少出错**地完成生产动作，而非视觉炫技。

本设计系统覆盖三个表面（桌面 / 一体机 / 移动），提供统一令牌 + 组件库，并以 VitePress 文档站承载设计哲学、令牌与逐组件文档。

## Brand Personality

**克制、精确、可信赖**。工业控制台气质，参照 Vercel / Linear 的克制工艺感：近黑深色基底、去饱和语义色、发丝级描边、克制动效。不喧哗、不卖弄；信息与状态一眼可辨，可长时间注视。

## Anti-references

- 通用 SaaS「奶油色 / 米白渐变 + 大字 hero + 营销 buzzword」落地页观感。
- 过度装饰的仪表盘：彩色大数字模板、堆叠卡片网格、玻璃拟态滥用、侧边色条、渐变文字。
- 消费级花哨/玩味风格、圆角过大（卡片 >16px）、装饰性手绘 SVG。
- "为了优雅"的浅灰正文（对比不足）。

## Design Principles

1. **产品优先**：产品 / UX / PM 视角 > 业务与行标 > 工程便利；别被现成组件限制，必要时大胆造新件。
2. **引导而非说明书**：用结构、层级、状态、微交互引导操作，不用成段解说；展示真实可读编号（WO- / WC-）而非占位。
3. **原版零改动、定制复制重建**：shadcn/reka 原版不改，定制一律复制重建为 Pro/Mobile 层 + 设计令牌；只走 `@nerv-iip/ui` 稳定导出边界，不深 import。
4. **克制的工艺感**：统一缓动令牌；拖拽要有跟手 + 橡皮筋 + 回弹；覆盖层克制玻璃质感，不过分。
5. **三表面一致、各自原生**：共享同一套令牌与语言，但各表面遵循其平台的原生手感与触控基线。

## Accessibility & Inclusion

- 对比度遵循 WCAG AA：正文 ≥4.5:1，大字 ≥3:1，placeholder 同样 4.5:1。
- 完整支持 `prefers-reduced-motion`：每个动画都有降级（淡入/瞬时）。
- 深色为基准但亮暗双主题；动态品牌色在等亮度 OKLCH 轮上选取，保证各色相一致的品牌强度。
- 移动端 ≥44px 触控目标、安全区适配。
